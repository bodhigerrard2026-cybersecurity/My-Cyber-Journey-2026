## The Kernel
<br>
The kernel is the core component of an operating system. It acts as a bridge allowing the hardware to communicate with the software. The kernel manages system resources such as the CPU, Memory, and peripheral devices. In short-term it controls everything that happens on your system.

# The Linux Shell

## What is the linux shell?

The linux shell is a powerful program that accepts your typed commands, and passes them into the operating system to execute.

## What is bash?

Bash stands for (Bourne Again Shell) program. Bash is the default shell for most linux. 

When you open a terminal, you are greeted by the shell prompt. It's apperance can vary between distrubutions, but it typically follows this format. username@hostname:current_directory$. Example: Bodhi@icloud:/home/projects$

The $ symbol at the end indicates that the shell is ready to accept commands from a regular user.
